# Eventually OKR

A comprehensive OKR (Objectives and Key Results) management application built with modern web technologies.

##  Overview

Eventually OKR is a full-stack application designed to help teams set, track, and manage their Objectives and Key Results effectively. The project consists of a NestJS backend API and a React TypeScript frontend.

##  Project Structure

```
Eventually-OKR/
├── eventually-backend/     # NestJS backend API
└── eventually-okr/         # React TypeScript frontend
```

##  Quick Start

### 1. Repository Setup

git clone https://github.com/prathamps/Eventually-OKR.git
cd Eventually-OKR

Navigate to the backend directory and follow the server configuration:

### 3. Frontend Setup

[ Frontend Documentation](./eventually-okr/README.md)
##  Tech Stack

- **Backend**: NestJS, Prisma, PostgreSQL
- **Frontend**: React, TypeScript, Vite
- **Package Manager**: pnpm
- **Containerization**: Podman

##  Prerequisites

- Node.js 18+
- pnpm
- Podman
